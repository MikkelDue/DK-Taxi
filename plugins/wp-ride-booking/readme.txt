== Description ==

WP Ride Booking Plugin is general purpose wordpress plugin for Ride Booking, Travel Agencies and Cab Booking Sites, Taxi Booking Sites who can directly accept payment from their paypal account.

There are following settings available in the plugin
*  Paypal Email Id
*  Google Map Api Key
*  Rates per kilo meter
*  Get Route Button Text
*  Set Google Map height
*  Currency select option
*  Set background color and text colors.

<h4>Key Features: </h4>
*  Calculates total distance between two locations.
*  Easy to use with shortcode provided.

<h4>Shortcode:</h4>
<pre>[wp-ride-booking]</pre>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Find the Plugin "WP Ride Booking" in Plugins and Activate it.
3. Use the Settings->WP Ride Booking to configure the booking settings.
